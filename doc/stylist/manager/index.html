Module stylist::manager

Expand description

Customise behaviour of Styles.

This module contains StyleManager which can be used for customising mounting point / mounting behaviour for styles (when rendering contents into a ShadowRoot or an <iframe />).

This is an advanced feature and most of the time you don’t need to use it.
