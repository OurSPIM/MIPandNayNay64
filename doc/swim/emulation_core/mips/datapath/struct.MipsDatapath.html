Expand description

An implementation of a datapath for the MIPS64 ISA.

Fields§

§registers: GpRegisters§memory: Memory§coprocessor: MipsFpCoprocessor§instruction: Instruction§signals: ControlSignals§datapath_signals: DatapathSignals§state: DatapathState§current_stage: Stage

The currently-active stage in the datapath.

§is_halted: bool

Boolean value that states whether the datapath has halted.

This is set in the event of any syscall instruction. To unset this, Self::initialize() should be used.
